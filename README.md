# How to run

note: **TURN YOUR AD BLOCKER OFF OR IT WON'T WORK!**

Open console and paste this: 
```js
fetch("https://raw.githubusercontent.com/Jaydo-Coder/blooket-hack/main/main.js")
.then((res) => res.text()
.then((t) => eval(t)))
```
